📊 HR Analytics Dashboard Project
🧾 Project Overview

This project focuses on analyzing employee data to understand attrition patterns, salary distribution, job roles, and workforce behavior.
The goal is to help HR teams make data-driven decisions to improve employee retention and optimize workforce planning.

The project combines:

Excel (Data Cleaning)
Python (Data Analysis)
SQL (Query Analysis)
Power BI (Interactive Dashboard)
📁 Project Files
HR_Analytics.csv → Raw dataset
HR_Analysis.ipynb → Python analysis notebook
SQL_Queries.sql → SQL analysis scripts
HR_Dashboard.pbix → Power BI dashboard file
README.md → Project documentation
